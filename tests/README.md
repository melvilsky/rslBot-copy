# Тесты

Эта папка содержит тестовые скрипты для разработки и отладки.

## Структура

- `test_live_arena_claim_refill.py` - тест для проверки claim_refill в Live Arena
- `screenshots/` - папка для скриншотов для тестирования

## Использование

### test_live_arena_claim_refill.py

Тестовый скрипт для проверки работы claim_refill в Live Arena.

**Как использовать:**

1. Сделайте скриншот экрана Live Arena (когда видна панель с квестами и наградами)
2. Сохраните скриншот в папку `tests/screenshots/` (например, `live_arena_screenshot.png`)
3. Запустите скрипт:
   ```bash
   python tests/test_live_arena_claim_refill.py
   ```
4. Выберите скриншот из списка
5. Скрипт покажет:
   - Проверку текущих координат claim_refill
   - Все красные точки в области правой панели квестов
   - Сохранит изображение с маркерами найденных точек

**Результат:**

Скрипт создаст файл `result_<имя_скриншота>` с визуальными маркерами:
- Синяя точка - текущие координаты claim_refill
- Красные точки - все найденные красные точки в области

## Важно

- Тесты не включаются в компиляцию (PyInstaller компилирует только main.py)
- Тесты можно оставлять в репозитории
- Скриншоты в `screenshots/` можно добавлять в .gitignore, если они большие
